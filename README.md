Tarot deck application built in Python using QT.

Current alpha functions allow cards to be pulled from a full deck, shuffled at launch. Double clicking on deck will pull the top card, face down. Double clicking cards will alternate facings. Cards can be freely dragged with left click and rotated with right click. Dragging a card onto the deck returns it as the top card.

Planned features:
-Shuffle during run time
-Pull specific cards from deck (important to locate signifier cards for readings)
-Helper pop up with reading instructions
-Helper pop up for card divinatory meanings

Rider-Waite images illustrated by Pamela Colman Smith and in the public domain, sourced from Wikimedia Commons.
